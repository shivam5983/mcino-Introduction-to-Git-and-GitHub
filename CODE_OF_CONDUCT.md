# Code of Conduct

## Our Pledge
We pledge to make participation in our project a harassment-free experience.

## Our Standards
Examples of behavior that contributes to a positive environment include respect and collaboration.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing standards.

## Scope
This Code of Conduct applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported to the maintainers.

## Enforcement Guidelines
Maintainers will follow fair corrective action.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
